### AX — AI Agent Experience Toolkit

A suite of open-source tools to make websites AI-agent-ready:

| Tool | What it does | Install |
|------|-------------|---------|
| [**ax-audit**](https://github.com/lucioduran/ax-audit) | Lighthouse for AI Agents — audit any site's AX readiness | `npx ax-audit https://your-site.com` |
| [**ax-init**](https://github.com/lucioduran/ax-init) | Generate the AX files your site is missing | `npx ax-init` |
| [**ax-cite**](https://github.com/lucioduran/ax-cite) | Web component for AI-extractable citation blocks | `npm i ax-cite` |
| [**ax-web-readiness**](https://github.com/lucioduran/ax-web-readiness) | AX audit dataset of the top 10K websites | [Hugging Face](https://huggingface.co/datasets/lucioduran/ax-web-readiness) |
